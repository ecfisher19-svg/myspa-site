AR Room Scanning

True AR scanning requires a native iOS app using Apple's RoomPlan SDK and LiDAR sensor (iPhone 12 Pro and later). The Swift code for this is ready to generate — it just needs to be compiled into a native app.

The full scanning pipeline outputs a dimensionally accurate CapturedRoom object which bridges directly into this 3D editor. Accuracy: ±1–2cm.

Coming in the iOS release of MySpa. For now, use the simulated scan or enter your room dimensions manually.
